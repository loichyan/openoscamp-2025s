# 🏕️ 2025 春夏季开源操作系统训练营

这里记录了 2025 春夏季开源操作系统训练营[^1]的学习成果，主要内容分散在不同的分支：

- **afit-dyn**: 让包含 `async fn` 的 `trait` 支持动态派发
- **blog**: 学习过程中记录的博客
- **local-executor**: 一个极简的单线程异步执行器
- **spiral-os**: 一个学习性的半成品内核
